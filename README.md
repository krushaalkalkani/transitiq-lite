# TransitIQ Lite

TransitIQ Lite is a Next.js 14 App Router demo for answering campus transit questions from an uploaded document image.

## Setup

```bash
npm install
cp .env.example .env.local
```

Add your OpenAI API key to `.env.local`:

```bash
OPENAI_API_KEY=your_openai_api_key_here
```

## Development

```bash
npm run dev
```

Open `http://localhost:3000`.
